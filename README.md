planning-poker
==============

Planning poker(wip).

Usage
------

install

```sh
npm install
```

start

```sh
npm start
```

admin
-----

1. Create a room
2. Define max and min values
3. Wait for users to join the room
4. Start planning
5. Finish planning


users
------

1. Join the room
2. Wait for start planning
3. Input a value
4. Show planning value
